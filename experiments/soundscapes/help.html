BASIC CONTROLS

  • A-Z/0-9 + MOUSE MOVE = Assign node to position.
  • CLICK = Preview sound / Select node / Drag selected nodes.
  • TAB = Cycle through Edit Modes (OSC, ADSR, FILTER, FX, MOD).
  • ARROW UP/DOWN or SCROLL = Adjust parameter value.
  • ARROW LEFT/RIGHT = Select parameter type.
  • SPACE = Toggle sustain (continuous play).
  • BACKSPACE = Delete selected/pressed nodes.
  • ESC = Stop all sounds + clear selection.
  • RIGHT CLICK = Remove single node.
  • CTRL/CMD + A = Select all nodes.
  • SHIFT + F = Fullscreen mode.
  • ? = Toggle help overlay.

ABOUT THE SYNTH

Soundscapes is a minimalistic JavaScript synthesizer. It transforms your screen into a playable instrument where you assign notes to keyboard characters and then perform with them, now with a full suite of sound-shaping tools.

Building Your Instrument

The primary goal of the synth is to turn your computer to multi oscillator keyboard instrument. By assigning alphanumeric keys (a-z, 0-9) to different positions on the screen, you are mapping them to specific frequencies. Once a key is assigned, you can play it like a piano and tune it to your likings.

  • Assign a Key: Move your mouse to a location, then press and hold a key like Q. A sound will play. When you release the key, a node labeled "Q" will be created at that position.
  • Play Your Instrument: Now, you can press the Q key at any time to play that exact sound, no matter where your mouse is.
  • Compose: Assign multiple keys like Q, W, E, R to create a musical scale or a chord, then play them with your keyboard.

Selection & Manipulation

Once you have nodes on the screen, you can select and modify them individually or as a group.

  • Select a Node: Simply click on any node to select it. It will glow cyan.
  • Multi-Select: Hold SHIFT or CTRL/CMD and click multiple nodes to add them to your selection. You can also use CTRL/CMD + A to select all nodes.
  • Drag Nodes: Click and drag a selected node to move it. If you have multiple nodes selected, they will all move together relative to the one you are dragging.

EDIT MODES

Press TAB to cycle through different modes for editing the selected nodes. The current mode is displayed in red at the bottom of the screen.

1. OSC MODE (Oscillator)

Controls the fundamental sound of a node.

  • ARROW LEFT/RIGHT = Cycle through oscillator waveforms (Sine, Triangle, Square, Sawtooth).
  • ARROW UP/DOWN or SCROLL = Adjust the node's master volume.

2. ADSR MODE (Envelope)

Shapes the volume of a node over time.

  • ARROW LEFT/RIGHT = Select the envelope parameter to edit.
  • ATTACK: How long it takes for the sound to reach full volume.
  • DECAY: How long it takes to drop to the sustain level.
  • SUSTAIN: The volume level while a key is held down.
  • RELEASE: How long it takes for the sound to fade out after the key is released.
  • ARROW UP/DOWN or SCROLL = Adjust the selected parameter's value.

3. FILTER MODE

Cuts or boosts frequencies to change the timbre of the sound.

  • ARROW LEFT/RIGHT = Cycle through filter types (Lowpass, Highpass, Bandpass, Notch).
  • ARROW UP/DOWN or SCROLL = Adjust the filter's cutoff frequency.

4. FX MODE (Effects)

Adds various audio effects to the sound.

  • ARROW LEFT/RIGHT = Select the effect parameter to edit.
  • DISTORT: Adds harmonic distortion for a gritty sound.
  • REVERB: Simulates the sound of a room or space.
  • ECHO MIX: Controls the volume of the echo repeats.
  • ECHO FDBK: Controls how many times the echo repeats (feedback).
  • TREMOLO: Modulates the volume to create a pulsating effect.
  • PAN: Positions the sound in the stereo field (left/right).
  • ARROW UP/DOWN or SCROLL = Adjust the amount of the selected effect.

5. MOD MODE (LFO Modulation)

Uses a Low-Frequency Oscillator (LFO) to create rhythmic changes to the sound.

  • ARROW LEFT/RIGHT = Select the modulation target.
  • FILTER: The LFO modulates the filter frequency (classic "wobble" effect).
  • PITCH: The LFO modulates the oscillator's pitch (vibrato effect).
  • ARROW UP/DOWN = Adjust the LFO Depth (how strong the modulation is).
  • SCROLL = Adjust the LFO Rate (how fast the modulation is).

VISUAL INDICATORS

  • Green Ring = Assigned but not playing.
  • Cyan Ring = Selected node.
  • Purple Ring = Sustained node.
  • Blue Inner Ring = Indicates a selected node is also playing or sustained.
  • Pulsing/Glowing Ring = The node is currently playing sound.
  • Gray Dashed = Empty placeholder for future assignment.

TROUBLESHOOTING

  • No sound? Click anywhere to initialize the audio context. The "Click to Connect" message will turn green.
  • Stuck notes? Press ESC to stop all sounds.
  • Too many nodes? ESC + BACKSPACE clears everything.
  • Fullscreen not working? Use SHIFT + F.
